Country Search App
Aplicación web interactiva que permite buscar países en español, mostrar información relevante y visualizar su ubicación geográfica en un mapa interactivo usando Leaflet y la API pública REST Countries.

Características principales
Búsqueda de países en español
Mapa interactivo centrado en el país
Visualización de la bandera
Información detallada:
Capital
Región y subregión
Población
Idiomas
Moneda
Zonas horarias

Diseño responsive con Bootstrap
Carga eficiente mediante cache local de países

Tecnologías utilizadas
HTML5
CSS3
JavaScript (ES6)
Bootstrap 5
Leaflet.js (mapas)
REST Countries API
OpenStreetMap

Instalación y uso
Clonar el repositorio
git clone https://github.com/tu-usuario/country-search-app.git

Abrir el proyecto
No requiere servidor.
Abre index.html directamente en el navegador.

Cómo usar la app
Escribe el nombre del país en inglés
Presiona el botón Buscar
Visualiza:
Información detallada del país
Mapa centrado en su ubicación
Mapas
La aplicación utiliza Leaflet.js con mapas de OpenStreetMap:
Zoom automático
Marcador con popup
Ajuste responsive
